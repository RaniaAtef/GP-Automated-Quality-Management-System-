Automated Quality Management System

The present work introduces a framework for an automated information system that manages the quality in our higher educational institution.

"Vision & Mission "

  • Vision

    Implementing software system will help the university to assessment of the current system which is used in the university. The system will allow users to edit information fields directly, data can easily transmitted and retrieved, faster data access that will increase productivity for workers. Replacing paper-based quality management automated system with automated quality software and that will improve the quality standards and knowledge management requirements, further decreasing time and cost when producing statistical reports.

  • Mission

    Replacing paper-based quality management system with automated quality software and that will improve the quality standards and knowledge management requirements, further decreasing time and cost when producing reports and making evaluations. The system will allow users to edit information fields directly, data can easily transmitted and retrieved, faster data access that will increase productivity for workers.
